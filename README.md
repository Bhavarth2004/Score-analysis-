# Score-analysis-
Project Overview This project analyzes a real-world marketing campaign dataset to identify patterns in customer behavior and predict lead conversion. By exploring demographics, spending habits, and past campaign interactions, this analysis provides actionable insights to optimize marketing spend and increase the overall conversion rate.   
Objectives
Data Cleaning: Handle missing values (e.g., Income) and filter outliers.

Conversion Analysis: Calculate the baseline conversion rate and identify high-performing segments.

Behavioral Profiling: Analyze how spending on products (Wine, Meat, etc.) correlates with a positive response.

Demographic Insights: Evaluate the impact of education level and household composition on lead quality.
